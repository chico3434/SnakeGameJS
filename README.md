# Snake Game
Jogo da cobrinha feito usando canvas do JavaScript.

## Jogar agora
É possível jogar clicando **[aqui](https://chico3434.github.io/SnakeGameJS/)**.

## Como jogar 
Basta usar as setas direcionais do teclado para alterar a direção da cobrinha e pegar as maçãs passando por cima delas.  
Não bata com a cabeça da cobrinha em outra parte do corpo dela, pois caso isso ocorra o jogo acaba.  
Caso o jogo acabe, há a opção de reiniciar o jogo no botão **Reiniciar**, que basicamente zera a pontuação e deixa a cobrinha do tamanho inicial.

### Alterar cores
É possível alterar as cores da cobrinha, da maçã e do fundo trocando os valores das caixas de seleção que se encontram abaixo do botão **Reiniciar**
